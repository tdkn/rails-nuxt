## Getting started

```
$ docker-compose up -d
```

or if you are using [`docker-sync`](http://docker-sync.io/):

> ```
> $ docker-sync-stack start
> ```

Take a look at [the documents](https://docker-sync.readthedocs.io/en/latest/) to learn more about usage.
